# Architecture Decision Records (ADR)

Registro de decisiones de arquitectura del proyecto **punto-facturacion**, en orden numérico.

| ID | Título | Estado |
|----|--------|--------|
| [0001](0001-platform-identity-model.md) | Modelo de identidad para operadores de plataforma | Aceptada |
| [0002](0002-platform-jwt-and-authorization.md) | JWT, claims y autorización (policies) para plataforma | Aceptada |
| [0003](0003-platform-api-host-and-surface.md) | Superficie HTTP y despliegue de la API de plataforma | Aceptada |

Convención: cada ADR describe **contexto**, **decisión**, **consecuencias** y qué fases de `docs/PLATFORM-ADMIN-FASES.md` la materializan en código.
